# Day 02 - pH Calculator

## Description

I wrote a Python program to calculate the pH of a chemical solution based on the hydrogen ion concentration.

The program uses the well-known chemistry formula:

pH = -log10[H+]

The program receives the hydrogen ion concentration in mol/L and returns the calculated pH value with two decimal places.

---

## How to Use

1. Run the script using Python  
2. Enter the hydrogen ion concentration in mol/L  
3. The program will calculate and display the pH value

---

## Example to Use

Suppose the hydrogen ion concentration is:

0.001 mol/L

Calculation:

pH = -log10(0.001)

pH = 3.00

Output:

The pH of the solution is: 3.00
